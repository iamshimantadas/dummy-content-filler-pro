Generate realistic dummy content for WordPress including posts, users, and WooCommerce products with automatic custom field population.

== Description ==

Dummy Content Filler Pro is a powerful plugin designed for developers and QA testers who need realistic test data inside WordPress.

The plugin can automatically generate dummy posts, users, and WooCommerce products while intelligently detecting and filling custom meta fields created using popular frameworks like ACF and CMB2.

It supports more than 100 realistic data types such as names, emails, prices, URLs, dates, and more, making your test environment look like a real production site.

The plugin also provides visual progress tracking while generating data and a safe one-click cleanup option that removes only the generated dummy content without affecting real data.

This makes it perfect for testing themes, plugins, WooCommerce stores, and site performance without spending hours manually creating content.

== Features ==

* Generate dummy posts in bulk
* Generate dummy users for testing user roles and permissions
* Generate WooCommerce products for store testing
* Automatic detection of ACF and CMB2 fields
* Populate 100+ realistic data types including names, emails, prices, dates, and URLs
* Automatically create and assign taxonomies
* Real-time progress tracking during generation
* Safe one-click cleanup for all generated content
* Removes generated post meta, user meta, and order meta
* Designed for developers and QA testers

== Installation ==

1. Upload the `dummy-content-filler-pro` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Navigate to the **Dummy Content Filler Pro** menu in the WordPress admin panel.
4. Configure the number of posts, users, or products you want to generate.
5. Click generate and monitor the real-time progress.

== Frequently Asked Questions ==

= Will this plugin delete my real content? =

No. The cleanup feature only removes the dummy content generated by the plugin.

= Does this plugin support WooCommerce products? =

Yes. If WooCommerce is installed, the plugin can generate dummy products for testing purposes.

= Does it support ACF custom fields? =

Yes. The plugin automatically detects and fills fields created using Advanced Custom Fields (ACF) and CMB2.

= Can I remove the generated data later? =

Yes. The plugin provides a safe one-click cleanup option to remove all generated dummy content.
