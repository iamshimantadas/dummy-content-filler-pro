# WP Dummy Content Filler

WP Dummy Content Filler is a powerful WordPress plugin designed for QA testers and developers to generate realistic dummy posts, users, and WooCommerce products. It intelligently detects and populates custom meta fields from ACF, CMB2, and other frameworks with 100+ data types.

Watch real-time progress with visual tracking and safely remove all generated dummy content with a single click. Perfect for testing themes, plugins, and site performance — without manual data entry.

---

## 🚀 Features

- Generate dummy posts, users, and WooCommerce products in bulk
- Automatically detect and populate ACF, CMB2, and custom meta fields
- Supports 100+ realistic data types (names, emails, prices, dates, URLs, and more)
- Automatically create and assign taxonomies to posts and products
- Safe one-click cleanup — removes only generated dummy content (including post meta, user meta, and order meta)

---

## 📦 Installation

### Method: WordPress Admin

1. Go to **Plugins → Add New**
2. Search for **"WP Dummy Content Filler"**
3. Click **Install Now**
4. Click **Activate**
